## 💎 Project Description & Concept
Moving away from standard, predictable corporate web templates, this project implements a custom-engineered **"Luxury Cyber-Noir"** single-page application structure. Built free of external CSS frameworks, it presents professional credentials through a dark-mode obsidian interface accented with warm metallic gold typography grids, automated layout animations, and an asymmetrical data feed.

## 🚀 Key Features
* **Premium Cyber-Noir Hero Panel:** Fixed-axis left sidebar showcasing primary contact matrices, active communication links, and a custom CSS slow-spinning animated profile image frame.
* **Dynamic Technical Matrix:** Interactive skills panel powered by native JavaScript event listeners, enabling real-time tag sorting and classification filtering (All Stack, Programming, Tools & Systems, Languages).
* **Chronological Experience Stream:** Sleek vertical timeline tracking peer mentoring achievements, internship tenures, and visual production roles.
* **Architecture Engineering Showcase:** Grid-aligned project portfolio blocks presenting recent software configurations, network protocols, and chatbot projects.
* **Ultra-Responsive Layout Engine:** Fluid grid parameters scaling seamlessly from ultra-wide desktop monitors down to standard mobile viewport displays.

## 🛠️ Technologies Used
* **HTML5:** Semantic, clean document mapping structures.
* **CSS3 Design Systems:** Root variable states, custom keyframe animation paths (`rotateClockwise`), and flex/grid alignment boxes.
* **Native JavaScript (ES6):** Dynamic DOM manipulation for interactive matrix data filtering.
